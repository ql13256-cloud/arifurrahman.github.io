A cybersecurity simulation portfolio featuring 12 industry-focused projects across healthcare, pharma, technology, automotive, finance, manufacturing, defense, and critical infrastructure.

## Highlights

- 12 interactive cyber defense simulations
- Industry-specific security scenarios
- Coverage of compliance, threat detection, anomaly detection, cloud security, IoT security, and incident response
- Built for portfolio, resume, and recruiter presentation
